# Lalsea Biorem Image Replace Guide

Trang review:

- `standalone/lalsea-biorem/index.html`

Thu muc asset:

- `standalone/lalsea-biorem/assets/`

Danh sach file dang duoc dung:

1. Hero packshot

- File: `assets/lalsea-biorem-packshot.svg`
- Goi y thay bang: packshot san pham PNG, JPG, WebP hoac SVG

2. Water dashboard visual

- File: `assets/module-water-dashboard.svg`
- Goi y thay bang: dashboard, bieu do NH3/NO2, anh report ky thuat

3. Bottom pressure visual

- File: `assets/module-bottom-pressure.svg`
- Goi y thay bang: anh day ao, bun day, anh hien truong, visual huu co

4. Application timing visual

- File: `assets/module-application-window.svg`
- Goi y thay bang: timeline ap dung, infographic quy trinh, flow xu ly

Luu y:

- Cach nhanh nhat la giu nguyen ten file va thay noi dung file.
- Neu dung ten file moi, cap nhat lai `src` trong `index.html`.
- Trang UI da duoc loai bo phan huong dan noi bo, nen file nay la noi tham chieu chinh de thay asset.
